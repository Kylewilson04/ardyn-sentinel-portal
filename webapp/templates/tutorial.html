Welcome to Ardyn 👋

🔒 What makes Ardyn different?

Ardyn is a private AI assistant that puts you in control. Unlike ChatGPT, Claude, or Gemini, your conversations are encrypted, processed through the sovereign destruction pipeline, and never used to train models.

Key difference: Every inference generates a cryptographic proof that your data was destroyed after processing.

⚡ Quick Start

  1. Complete your profile

    Tell Ardyn about your preferences so it can assist you better. This information is encrypted and never shared.

  2. Start a conversation

    Go to Chat and select an AI model. All 5 models run on your private infrastructure.

  3. Import your history (optional)

    Bring conversations from OpenAI, Claude, or Gemini. Ardyn will learn your style without storing the raw data.

🎓 What you'll learn in this tutorial

This guide covers Ardyn's key features: encrypted chat, destruction proofs, model selection, history import, and API access. Use the tabs above to explore each topic.

Chat Features 💬

💭 Conversation Sidebar

All your conversations are listed in the left sidebar. Click any conversation to continue where you left off. Conversations are automatically saved and encrypted.

// Your conversations are encrypted server-side You: What's the weather like? Ardyn: I don't have access to real-time weather data, but I can help you find a weather service or discuss climate topics.

🤖 Model Selection

Choose from 5 AI models optimized for different tasks:

  • Llama 3.3 70B — Balanced performance for general tasks
  • DeepSeek R1 70B — Deep reasoning and analysis
  • Qwen3 Coder 30B — Code generation and technical tasks
  • Nomic Embed Text — Text embeddings and semantic search
  • all-MiniLM — Fast, lightweight embeddings
Tip: Switch models mid-conversation to compare responses or use the best model for specific tasks.

📝 Message Actions

Hover over any message to see actions:

  • Edit — Change your message and regenerate the response
  • Copy — Copy message text to clipboard
  • View Proof — See the cryptographic destruction proof

Privacy & Security 🔐

🛡️ Zero Trace Protocol

Every conversation on Ardyn follows the Zero Trace Protocol:

  1. Encrypt

    Your message is encrypted before leaving your browser

  2. Process

    Processed within the sovereignty boundary

  3. Destroy

    Data is cryptographically shredded after inference

  4. Prove

    You receive a signed certificate of destruction

📜 Destruction Proofs

Every inference generates a Death Certificate — cryptographic proof that your input data was processed and destroyed. View your proofs in the Proofs dashboard.

{ "job_id": "ads-job-001", "destruction_nonce": "a3f7d2e8...", "zeroization_proof": "merkle_root_hash", "timestamp": "2026-02-16T22:30:00Z", "enclave_attestation": "signed_by_tee" }
Important: These proofs are verifiable by third parties. Your privacy is mathematically guaranteed, not just promised.

🔒 Encrypted Storage

Your conversations and profile are encrypted using AES-256-GCM. The encryption keys are derived from your user ID and never stored in plaintext.

AI Models 🧠

🏠 Private Infrastructure

All AI models run on your private infrastructure. Currently hosted on Azure Confidential Computing's H200 TEE (Trusted Execution Environment) — a secure enclave that ensures:

  • Your data is never visible to the hosting provider
  • Models cannot be tampered with
  • Cryptographic attestation proves integrity

Import History 📥

📤 Bring Your Conversations

Import chat history from OpenAI, Claude, or Gemini. Ardyn will:

  1. Analyze your style

    Extract communication patterns, topics, and preferences

  2. Create your profile

    Build a personalized profile (the raw data is discarded)

  3. Import conversations

    Store imported chats in your encrypted vault

Privacy note: Raw import data is processed and immediately discarded. Only the derived profile and your chosen conversations are kept.

📋 How to Export from Other Platforms

OpenAI: Settings → Data Export → Download conversations.json

Claude: Settings → Account → Export Data

Gemini: Google Takeout → Select Gemini Apps data

Personal Concierge & iPhone Reminders 🎯

🎯 Personal Concierge Persona

The Personal Concierge persona helps with daily tasks, planning, scheduling, and can create reminders that sync directly to your iPhone!

How to use: In Chat, select the 🎯 Personal Concierge from the persona dropdown, then ask for reminders naturally.

📱 iPhone Reminders Integration

Connect Ardyn to your iPhone Reminders app for seamless task management.

One-Time Setup:

  1. Install Shortcuts App

    Make sure Apple's Shortcuts app is installed (free on App Store)

  2. Create "Ardyn Reminder" Shortcut

    Open Shortcuts → Tap + → Name it exactly "Ardyn Reminder"

  3. Add Actions

    Add: 1) Receive Input (Text), 2) Get Dictionary from Input, 3) Add Reminder

  4. Done!

    Now when you ask the Concierge for reminders, you'll get a button to add to iPhone

💬 Example Commands

Try these with the Personal Concierge:

  • "Remind me to call mom tomorrow at 3pm"
  • "Add 'buy groceries' to my reminders for Saturday"
  • "Set a daily reminder to take my medicine at 8am"
  • "Remind me about the meeting in 2 hours"
// Ardyn will respond with a clickable button: I'll set that reminder for you! 📱 Add to iPhone Reminders

🔔 How It Works

When you ask the Personal Concierge to set a reminder:

  1. AI Understands

    The Concierge parses your natural language request

  2. Creates Reminder

    Ardyn generates a reminder object with title, time, and priority

  3. iOS Shortcuts Link

    A special URL is created that opens the Shortcuts app

  4. One Tap Add

    You tap the button and the reminder is added to your iPhone

Privacy: Reminder data is stored encrypted in your Ardyn account. The iPhone integration uses Apple's secure Shortcuts protocol.
